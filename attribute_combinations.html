Test Website - Attribute Combinations

Elements Structure Notes
A <a href=example onclick=index>
</a> 		Anchor tag with 2 attributes: href, onclick
B <a onclick=index href=example>
</a> 		Anchor tag with 2 attributes: onclick, href (order swapped)
C <a href=example onchange=index>
</a> 		Anchor tag with 2 attributes: href, onchange
<input type=checkbox onclick=example onchange=index> Checkbox with 2 attributes: onclick, onchange
<select onclick=example onchange=index>
<option value=Stay>
<option value=Go>
</select> 		Select with 2 attributes: onclick, onchange
F <a href=example onmouseenter=index>
</a> 		Anchor tag with 2 attributes: href, onmouseenter
<select onmouseenter=changeSelection onchange=index>
<option value=hello>
<option value=bye>
</select> 		Select with 2 attributes: onmouseenter, onchange. Hovering above the select does not trigger the onchange event.

H - This is a text to hover over.

 <p onmouseenter=example onmouseover=index>
</p> 		Paragraph with 2 attributes: onmouseenter, onmouseover
