Attribute Combinations

Elements Structure Notes
A <a href=example onclick=alert>
</a> 		Anchor tag with 2 attributes: href, onclick
B <a href=example onchange=alert>
</a> 		Anchor tag with 2 attributes: href, onchange
C <a href=example onmouseenter=alert>
</a> 		Anchor tag with 2 attributes: href, onmouseenter. The href cannot be clicked after moving the mouse here, because an alert opens.
D This is a secret text! <a href=example onmouseenter=toggletext>
</a> 		Anchor tag with 2 attributes: href, onmouseenter. The href can still be clicked after moving the mouse here because it just causes text to appear. It leads to the example website.
